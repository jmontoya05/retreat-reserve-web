# Retreat Reserve — Plataforma Premium de Aliler de Cabañas 🌲

Bienvenido al frontend de **Retreat Reserve**, una aplicación web premium diseñada para explorar, calificar y reservar exclusivas cabañas rodeadas de naturaleza (bajo un diseño inspirado en la paz del bosque y un fondo blanco de alta definición). 

Este proyecto fue desarrollado en **React 19 + TypeScript + Vite + Tailwind CSS**, garantizando un alto rendimiento y apego estricto a las mejores prácticas de arquitectura estática (Clean Architecture adaptada a React y división modular de componentes).

---

## 🚀 Comandos Rápidos del Proyecto

### 1. Iniciar en modo desarrollo
Inicia el servidor local en el puerto `3000`.
```bash
npm run dev
```

### 2. Compilar el proyecto para producción
Bundling y optimización de estáticos en la carpeta `dist/`.
```bash
npm run build
```

### 3. Verificar tipados y sintaxis
Ejecuta el linter estático de TypeScript.
```bash
npm run lint
```

## 🎨 Paleta de Colores Naturales e Identidad Visual
En armonía con las directrices de la marca, implementamos una interfaz limpia:
- **Base Principal**: `#FFFFFF` (Fondo Blanco de alta legibilidad para relajar la visual).
- **Verde Primario / Sea Green**: `#2E8B57` (Para botones principales y estados activos).
- **Verde Profundo / Deep Forest**: `#1F5937` (Para pie de página y cabeceras).
- **Acento Oliva**: `#8DB600` (Para insignias, etiquetas de características y puntuaciones).
- **Warm Sand / Arena Cálida**: `#F4E9D9` (Para contenedores secundarios, llamadas y fondos suaves).
- **Texto Neutral**: `#1F2937` (Para excelente contraste según estándar WCAG AA).
